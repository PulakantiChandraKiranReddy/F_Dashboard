# 📊 Financial Dashboard - Daily Use Guide

## 🚀 Quick Start

### 1. **Dashboard Overview** (`/dashboard`)
- **Real-time data**: All charts and numbers update automatically
- **Three main tabs**:
  - 📊 **Overview**: Financial health summary, monthly performance
  - 💸 **Expenses**: Detailed expense analytics with charts
  - 💰 **Income**: Income tracking and source analysis

### 2. **Adding Transactions** (`/admin`)
- **Expense Form**: Add daily expenses with categories
- **Terminal Commands**: Quick data entry via command line
- **Real-time Updates**: Changes appear instantly across all views

## 📈 Key Features for Daily Use

### **Financial Health Monitoring**
- **Savings Rate**: Automatically calculated (aim for 20%+)
- **Expense Ratio**: Track spending vs income (keep under 80%)
- **Monthly Performance**: Current month vs total trends
- **Real-time Status**: Live indicators show data freshness

### **Expense Analytics**
- **30-Day Trend**: Line chart showing daily spending patterns
- **Category Breakdown**: Pie chart of spending by category
- **Recent Transactions**: Quick view of latest expenses
- **Monthly Comparison**: This month vs previous months

### **Income Tracking**
- **12-Month Trend**: Area chart showing income over time
- **Source Analysis**: Bar chart of income sources
- **Monthly Averages**: Automatic calculation of average monthly income
- **Income History**: Detailed table of all income entries

## 🎯 Daily Workflow

### **Morning Check** (5 minutes)
1. Open `/dashboard` → Overview tab
2. Check monthly performance vs budget
3. Review savings rate and financial health
4. Note any unusual spending patterns

### **Adding Expenses** (2 minutes)
1. Go to `/admin` → Expense Form
2. Fill in: Title, Amount, Category, Date, Notes
3. Submit → Data appears instantly in dashboard

### **Quick Commands** (1 minute)
Use Terminal in `/admin` for fast entries:
```bash
# Add expense
add expense 1500 Food Grocery shopping

# Add income  
add income 50000 Salary Monthly salary

# Check balance
show balance

# View recent transactions
show expenses
show income
```

### **Evening Review** (3 minutes)
1. Check `/dashboard` → Expenses tab
2. Review daily spending trend
3. Analyze category breakdown
4. Plan tomorrow's budget

## 📊 Understanding the Charts

### **Overview Tab**
- **Green Cards**: Income and positive metrics
- **Red Cards**: Expenses and spending
- **Blue Cards**: Net savings and performance
- **Purple Cards**: Transaction counts and activity

### **Expenses Tab**
- **Line Chart**: 30-day spending trend (red line)
- **Pie Chart**: Category breakdown with percentages
- **Table**: Recent expenses with dates and amounts

### **Income Tab**
- **Area Chart**: 12-month income trend (green area)
- **Bar Chart**: Income sources (horizontal bars)
- **Table**: Recent income with sources and dates
